# PL/SQL Exercises

This branch contains the submitted PL/SQL assignment work for:

- Exercise 1: Control Structures
- Exercise 3: Stored Procedures

## Folder Structure

```text
PLSQL_Exercises/
|-- Database/
|   |-- schema.sql
|   |-- sample_data.sql
|   |-- bank_schema.sql
|   |-- sample_bank_data.sql
|   |-- employee_schema.sql
|   `-- sample_employee_data.sql
|-- Exercise-1/
|   `-- ControlStructures/
|       |-- Scenario-1/
|       |-- Scenario-2/
|       `-- Scenario-3/
`-- Exercise-3/
    `-- StoredProcedures/
        |-- Scenario-1/
        |-- Scenario-2/
        `-- Scenario-3/
```

## How To Run

For Exercise 1, run:

1. `Database/schema.sql`
2. `Database/sample_data.sql`
3. The scenario SQL file under `Exercise-1/ControlStructures`

For Exercise 3, run:

1. `Database/bank_schema.sql`
2. `Database/sample_bank_data.sql`
3. `Database/employee_schema.sql`
4. `Database/sample_employee_data.sql`
5. The scenario SQL file under `Exercise-3/StoredProcedures`

Each scenario folder includes its SQL solution, README, and captured output.
